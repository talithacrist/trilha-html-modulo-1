Tags

Tags são elementos de marcação que definem o conteúdo de uma página da WEB. Elas são escritas entre colchetes angulares < >, geralmente vem em pares: uma de abertura, e outra de fechamento. As tags vão indicar ao navegador como exibir o conteúdo: títulos, parágrafos, imagens, links. Elas são a base da linguagem HTML. Algumas tags não necessitam de barra de fechamento, como a input e img. E algumas tags são auto-fecháveis, como a < br > (quebra de linha).

Lembre-se:
É fundamental manter a estrutura do HTML para o melhor funcionamento e semântica para a acessibilidade e manutenção do código.
É importante fechar todas as tags, menos as tags auto-fecháveis, para evitar erros de exibição.

1. Atibutos da Tag

Atributos são propriedades que podem ser adicionadas às tags, com a finalidade de modificar o seu comportamento ou aparência.
Elas são especificadas nas tags de abertura, e geralmente consistem em um nome + um valor, por exemplo: < img src="imagem.png" alt="Descrição da imagem"> . Nesse exemplo, os atributos utilizados na tag < img > foram src, que define a imagem e alt, que define a descrição da imagem colocada.

Algumas exemplos de propriedades são:

  • id: identificador do elemento

  • style (CSS): colocar o comando direto no elemento

  • class (CSS): padronizar o elemento

  • input type (mais usados): especifica qual tipo de dado o programador deseja inserir
    Exemplo: < input type="text" name="nome" placeholder="Seu nome" >

  • href: usado para colocar links

2. Textos (tipografia)

As tags de texto são usadas para formatar e estruturar o conteúdo em páginas da WEB, ele define como o conteúdo vai se portar. São elementos importantes e fundamentais para a formatação e apresentação visual do conteúdo. Elas se dividem em tags de formatação e tags que definem elementos semânticos.

São alguns exemplos de tag de texto:

  • h: header (cabeçalho, título) varia de h1 a h6, sendo h1 o maior, e h6 o menor tamanho.

  • p: parágrafos

  • strong: negrito

  • i: itálico

  • u: underline

  • mark: marca texto

  • blockquote: citação

3. Listas

No HTML, as tags de listas são usadas para criar listas ordenadas e não ordenadas. As principais listas são < ul >: listas não ordenadas (possuem marcadores), e < ol >: listas ordenadas, acompanham numeros ordinais, numeradas. e a tag < li > usada para cada item da lista.

LISTA NÃO ORDENADA

  • A tag < ul > representa uma Lista não ordenada
  • Cada item da lista acompanha a tag < i > dentro da tag principal (ul)
  • Normalmente os itens são exibidos com marcadores como pontos, circulos.
Código de lista não ordenada:
< ul >
     < li >Item 1 < / li >
     < li >Item 2 < / li >
     < li >Item 3 < / li >
< / ul>

LISTA ORDENADA

  • A tag < ol > representa uma Lista ordenada
  • Cada item da lista acompanha a tag < i > dentro da tag principal (ol)
  • Cada item da lista é representada por uma sequência numérica
  • É possivel alterar o tipo de ordenação (números romanos, letras, etc) com a tag de atributo type
Código de lista ordenada:
< ol >
     < li >Item 1 < / li >
     < li >Item 2 < / li >
     < li >Item 3 < / li >
< / ol>
Ou:
< ol type="a" >
     < li >Item a < / li >
     < li >Item b < / li >
     < li >Item c < / li >
< / ol>
